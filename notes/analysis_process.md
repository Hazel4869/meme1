# 归纳式主题分析流程记录

## 1. 数据准备

- 用户提供两份 Word 逐字稿。
- 使用本地文档读取工具抽取文字，生成 `data/raw/interview_01.txt` 与 `data/raw/interview_02.txt`。
- 为降低伦理风险，原始逐字稿不纳入 Git；公开项目保留匿名化材料清单和分析输出。

## 2. 初读与备忘

初读重点关注：

- meme 如何进入受访者视野；
- 无语佛在旅行决策中的权重；
- 线上图像与线下实物经验之间的差异；
- 拍照、分享、二创等后续行为；
- 博物馆与城市文旅可能受到的影响。

## 3. 开放编码

采用 inductive thematic analysis，不预设理论框架。先逐段标记可重复出现或与研究问题高度相关的意义单元，再合并为 16 个初始代码。

## 4. 主题聚合

将代码聚合为五个初步主题：

1. 从表情包到朝圣理由
2. 实体求证
3. 参与式再生产
4. 流量的双重效应
5. 可持续回应

## 5. 下一步建议

- 将剩余 17 份访谈加入同一代码本，检查是否出现新代码。
- 对不同群体做比较：内容创作者/普通游客、艺术背景/非艺术背景、高频旅行者/低频旅行者。
- 分析机构视角资料：馆方公告、文创产品、动线调整、社交媒体回应。
- 在论文写作前进行二轮编码，区分“动机”“体验”“再传播”“机构影响”“策略回应”五个分析层级。
